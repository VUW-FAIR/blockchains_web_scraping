Github: 
Format for each repository : 
all_contributors: json files of all contributors
branches: json file of all branches
comments_: json files of all comments 
commits: json files of all commits 
contributors: individual json files for each contributor (their github information)
issues: json files for issues 
pull_requests: json files for pull requests 
releases: json file for releases 
repo_info_: json file for repository information

JSON format pretty self explanatory as everything is named. More info can be found:
https://developer.github.com/v3/
